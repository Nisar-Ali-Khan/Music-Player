# 🎵 Music Player - Web-Based Audio Player

A modern, feature-rich web-based music player with playlist functionality, search, categories, and full audio controls. Built with vanilla HTML, CSS, and JavaScript.

## 🌟 Features

### Core Features
- **Play/Pause** - Control music playback
- **Next/Previous** - Skip between tracks
- **Volume Control** - Adjust volume with slider
- **Progress Bar** - Click anywhere to seek through the song
- **Current Time & Duration** - Display timing information

### Playlist Features
- **Dynamic Playlist** - View all available songs
- **Now Playing Highlight** - Active song is highlighted
- **Click to Play** - Select any song from the list

### Search & Filter
- **Search Songs** - Filter by title or artist name
- **Category Filters** - Browse by genre:
  - All Songs
  - Pop
  - Rock
  - Jazz
  - Electronic
  - Classical

### Advanced Features
- **Shuffle Mode** - Random playback order
- **Repeat Mode** - Loop current song
- **Auto-Play Next** - Automatically plays next song when current ends

### Design Features
- **Modern UI** - Gradient backgrounds and glassmorphism effects
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Smooth Animations** - Hover effects and transitions
- **Album Art Display** - Visual representation of current track

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure and audio element |
| CSS3 | Styling, animations, responsive design |
| JavaScript (ES6+) | Core functionality, DOM manipulation |
| Font Awesome | Icons for controls |
| Google Fonts | Poppins typography |
